# Security Policy

## Supported Versions

Only the `main` branch is currently receiving security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please DO NOT open a public issue. Instead, send an e-mail to the repository maintainer. 

We will strive to acknowledge your report within 48 hours and provide a timeline for the resolution.
